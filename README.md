#  Fable Desktop

**Claude Fable Desktop** is a lightweight native client for interacting with large language models locally and through cloud APIs. Built for speed and simplicity, it offers a distraction-free environment for chat, code generation, and document analysis.

## Core Features
- Multi-model support with local and remote providers
- Real-time streaming responses with low latency
- Built-in prompt library and template system
- Conversation branching and version history
- Dark mode and fully customizable UI themes

## System Requirements

### Supported Platforms
- **Windows 10 / 11**
- **macOS Ventura and later**
- **Linux (x86_64, aarch64)**

### Notes
- Internet connection required for cloud-based models
- Local models require a compatible GPU or NPU
